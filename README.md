# SimpleBS 
## 自制的网络测试小工具
## A Simple Network Ping and Traceroute Tool

  1、软件初步完成，没有使用Autolayout，UI上还有些bug，比如在运行过程中滚动textview会阻塞ping操作导致时间不准，文本长度过长时没能自动往上滚动，UI设计比较糙。<br>
  2、更多功能等待加入，欢迎大家在issue栏发言，谢谢大家！
